new creates an object on the spot when you call it
new is an operator! 

function Person(firstname, lastname){  
console.log(this) //--> points to the new object"  
this.firstname = firstname;  
this.lastname = lastname;  
}

var john = new Person('John', "Doe");  
var jane = new Person('Jane', 'Doe');

when you instantiated a person like this, "new" creates an object.

all the "this" references inside the Person function reference the "new" object

constructing objects using functions.

a Function Constructor - a normal function that is used to construct objects

when you put a new keyword in front of a function constructor the this keywords point to the new empty object and that object is pointed at automatically
