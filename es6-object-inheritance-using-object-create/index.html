ES6 Object Inheritance using Object.create

...
var person {  
  firstname: 'Default',
  lastname: 'Default',
  greet: function(){
      return 'Hi' + this.firstname;
  }
}

var john = Object.create(person);

john.firstname = 'John';  
john.lastname = 'Doe';

ES6 and classes

a javascript class

class Person {
constructor(firstname, lastname){ this.firstname = firstname; this.lastname = lastname; } greet () {
return 'Hi ' + firstname; } }

var john = new Person('John', 'Doe');
//try this out! john will have the greet method and first name and last name properties!
